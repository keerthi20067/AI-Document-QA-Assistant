# AI Document QA Assistant

## Description

AI Document QA Assistant is an intelligent application that allows users to upload documents and ask questions about their content. The system uses Large Language Models (LLMs) to understand the document and provide accurate answers.

## Features

* Upload and analyze documents (PDF/Text)
* Ask questions related to the uploaded document
* AI-powered answers using LLM
* Interactive chat interface
* Easy-to-use UI built with Streamlit

## Tech Stack

* Python
* Streamlit
* LangChain
* OpenAI API
* FAISS (Vector Database)

## Installation

1. Clone the repository
   git clone https://github.com/yourusername/AI-Document-QA-Assistant.git

2. Navigate to the project folder
   cd AI-Document-QA-Assistant

3. Install required dependencies
   pip install -r requirements.txt

4. Run the application
   streamlit run app.py
